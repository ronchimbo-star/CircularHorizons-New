# Circular Horizons International

A modern, responsive website for Circular Horizons International - a pioneering sustainability consultancy dedicated to accelerating the transition to a circular economy.

## Features

- **Home Page** with hero section and four main service sections:
  - Consultancy Services
  - Green Registry Platform
  - ESG Reporting Platform
  - Mizan ESG

- **About Page** with mission, vision, values, and team information
- **Contact Page** with contact form and business information
- **News & Insights** page
- **Additional Pages**: Careers, FAQ, Privacy Policy, Terms of Service, Cookie Policy

## Technology Stack

- **Astro** - Fast, modern static site generator
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design

The website features a clean, professional design with:
- Green primary color (HSL: 142, 47%, 33%) representing sustainability
- Responsive layout that works on all devices
- Smooth scrolling navigation
- Professional typography and spacing
- High-quality imagery from Unsplash
